# CSCI497_Biometric_Authentication_App
Ajax Webpage that utilizes facial recognition for user detection and passkeys for authentication.

This program was written as a capstone project for CSCI 497 at Davenport University.

Demonstration video
https://youtu.be/1lShSKzrjmE?si=KE_NzD9anJIXYZLr&t=914

A demo of the front end of the site can be tested on github pages. However, the program will need to be run locally for the facial recognition models to function.
